blood line

Blood Bank is a project that aims to connect blood donors with hospitals and medical centers. The idea evolves around providing fresh blood for patients based on need and on time. The project also raises awareness about the benefits of blood donation, as well as include a rewarding system that encourages people to donate blood. This aims to have a large access to donors around the area of the origin of the request. This is my capstone project. Click here to download > project proposal > Implementation report > project completion report

features

  • Create & book appointmnet
    • Donors can search for appointmnets created by medical center on their desired day and time.
  • Create & book campaign
    • Donors can view nearby campaigns created by medical center and join others donating for a cause.
  • Admin Panel
    • Manage institutes account verfication.
  • SMS Notification
    • Donors will be notified preoir ro their booked appointments through a SMS as well when an emergency active campaign is listed nearby
  • Hospital Locator
    • Donors can view nearby hospitals based on their geo-location tag.
  • Chatbot on Facebook
    • Access all essential features of the website on messenger by chatting with a fully automated chatbot.

This project was handed to me first thing when I was employed at Moonline Travel. The project purpose is to provide a mean for the employees to log their item sales within the company. This allows the company to follow money transactions within branches and keeping entries logged for future data mining. For this project to work, I had to sit with multiple employees and observe their work pattern to create a web app that satisfies all teams working habits. This is to achive company goal while keeping a healthy work environemnt. Over the time, this project became high priority to me personally since there are people working around the clock using this system. Knowing how frustrated one can be, I came up with fun implementation from time to time to change employees routine and add a little bit of 'fun' into their work. I have thus so far added an animated login screen and a dark mode.

features

The app in biref is a web application that communicates with an API provided through SAP integration. This app collect user input, handles them for error checks and submits them to the API. There the app listens for response where error handling is active for a success or a fail message. The app closely mnitors every transaction by saving them into a databse. There is a login system to identify users with their levels for customized feature access, branch for seperating each branch work and identification.

Implmenting this application helped me go though understanding various type of users and their frustration. There were times where I never thought of implementing handlers for user inputs ( thought they wont make mistakes ) , Some disliked the presentation, others found difficulities in understanding the system. In short, the experince gained from the ongoing implemtation on this application is priceless.

Password Encryption

In an effort by managment to cut on payment, I was tasked to create a project tailored to our company needs that stores passwords securely. This project brought new challenges with it. There had to be a linux based server setup and encryotion placed to be on the safe side. I had setup an ubuntu server and went on searching for an encryption library that works. I then came up with Halite a PHP library that works with libsodium. This eased the encryption part of the project. After managing to have it implemented on our local linux server, I setup a Database that monitors every user action on this system. Part of making this system ours, is to restrict user access to the website after a login for about 60 seconds. To prevent employees from mistakenly leaving their passwords exposed around customers. Later, I implemented a Google sign in linked to our company's domain which is followed by a two step verfication via a chatbot on Facebook ( users were verfied to use that bot )
